# Cactus Multimedia - Galería

## Identidad
- **Nombre:** Cactus Multimedia - Galería
- **Ubicación:** Chiapas, México
- **Tono:** elegante, cálido, profesional — refleja la calidez del sureste mexicano con sofisticación
- **Idioma:** español

## Estilo visual
- **Paleta:** fondo oscuro minimalista con acento cálido terracota/ámbar como color principal
- **Inspiración:** wabi sabi — belleza en la simplicidad, texturas orgánicas, espacio negativo generoso
- **Tipografía:**
  - Títulos: serif elegante (Playfair Display o similar)
  - Cuerpo: sans-serif limpia (Inter, system-ui o similar)
- **Fotografía:** gran protagonismo visual, imágenes a todo ancho, transiciones suaves

## Tecnología
- HTML, CSS y JavaScript puro — sin frameworks ni dependencias externas
- CSS Grid y Flexbox para layouts
- CSS custom properties para la paleta de colores
- Sin librerías JS — Vanilla JS solo donde sea necesario (lightbox, menú móvil, scroll suave)
- Responsive design con mobile-first

## Categorías de contenido
- Retratos
- Quinceañeras

Cada categoría tendrá su propia galería con grid de imágenes. Las fotos reales se integrarán después; usar placeholders con relación de aspecto adecuada por ahora.

## Principios de diseño
1. Las imágenes son el centro — la interfaz no debe competir con ellas
2. Espacio negativo amplio — menos elementos, más respiración
3. Transiciones sutiles — nada estridente ni abrupto
4. Wabi sabi: texturas orgánicas, bordes suaves, imperfección controlada
5. Performance: carga rápida, sin dependencias externas
6. Accesibilidad: contraste adecuado, navegación por teclado, textos alternativos
